## Day 3 – 2026-04-02

### 🧠 Deep Learning
- Studied Sparse Autoencoders
- Key idea: enforce sparsity constraint on hidden layers to learn meaningful representations

### ⚙️ Concepts Covered
- Difference between regular autoencoders vs sparse autoencoders
- Use of L1 regularization / KL divergence for sparsity
- Activation sparsity: only few neurons active at a time

### 🤖 ML Insight
- Sparse representations improve feature extraction
- Helps in dimensionality reduction and anomaly detection

### 📝 Notes
- Overcomplete representations + sparsity = powerful encoding
- Widely used in unsupervised learning scenarios
